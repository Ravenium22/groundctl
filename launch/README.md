# Launch Assets

This folder is intentionally split into public and private launch material.

## Public (`launch/public`)

Content safe to keep in the open-source repository:

- blog drafts and case studies
- roadmap content
- assets intended for community consumption

## Private (`launch/private`)

Operational launch docs kept local:

- posting schedules and checklists
- outreach drafts (HN/Reddit/X/Product Hunt)
- application templates and internal audits

`launch/private/` is ignored via `.gitignore` by default.
